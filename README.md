# Frontend for flask app g1 deploy

-This is a base template for Vue3 projects. It includes the following:

- - Vue3
- - Vue Router
- - Vuex
- - i18n
- - Jest
- - SASS
- - Bootstrap 5

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
